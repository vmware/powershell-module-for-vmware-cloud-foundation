# Get-VCFLicenseMode

## Synopsis

Retrieves the current license mode.

## Syntax

```powershell
Get-VCFLicenseMode
```

## Description

The Get-VCFLicenseMode cmdlet retrieves the current license mode.

## Examples

### Example 1

```powershell
Get-VCFLicenseMode
```

This example shows how to retrieve the current license mode.
