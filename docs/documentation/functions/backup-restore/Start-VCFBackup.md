# Start-VCFBackup

## Synopsis

Starts the backup task in SDDC Manager.

## Syntax

```powershell
Start-VCFBackup
```

## Description

The Start-VCFBackup cmdlet starts backup task in SDDC Manager.

## Examples

### Example 1

```powershell
Start-VCFBackup
```

This example shows how to start the backup task in SDDC Manager.
