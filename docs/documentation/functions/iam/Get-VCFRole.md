# Get-VCFRole

## Synopsis

Retrieves a list of roles.

## Syntax

```powershell
Get-VCFRole
```

## Description

The `Get-VCFRole` cmdlet retrieves a list of roles from SDDC Manager.

## Examples

### Example 1

```powershell
Get-VCFRole
```

This example shows how to retrieve a list of roles from SDDC Manager.
