# Get-VcfProxy

## Synopsis

Gets the proxy configuration for the SDDC Manager.

## Syntax

```powershell
Get-VcfProxy
```

## Description

The `Get-VcfProxy` cmdlet retrieves the proxy configuration of the SDDC Manager.

## Examples

### Example 1

```powershell
Get-VcfProxy
```

This example shows how to get the proxy configuration of the SDDC Manager.
